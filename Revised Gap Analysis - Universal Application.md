**Purpose:** Meta-framework for escalating response intensity based on resistance level  
**Note:** This is a process framework, not a standalone handler. Can be referenced in advisor notes across multiple handlers.

**Script:**

```
</details>

<details>
<summary><b>LEVEL 2 - Moderate Resistance (3-4 deflections)</b></summary>

```

<details>
<summary><b>LEVEL 3 - Heavy Resistance (5+ deflections or hostility)</b></summary>

**Use:** Proof-first offer or polite disqualification

```
"I hear you - you need proof before we talk further. Here's what makes sense: I'll send 
you case studies and references today. You review on your timeline, and if it checks out, 
we reconnect. Sound good?"
```

**Option B - Polite Disqualification:**
```

we reconnect in [timeframe] if it makes sense?"
```
</details>

**Pattern Interrupt Techniques (Can be integrated into Level 2):**

<details>
<summary><b>Technique 1 - Meta-Conversation</b></summary>

```
"I notice we keep coming back to the same question. Let me address it directly: [answer]. 
**Does that resolve it, or is there something else you need to hear?**"
```
</details>

<details>
<summary><b>Technique 2 - Reverse the Question</b></summary>

```
"You keep asking about [topic]. **What specifically would need to be true about [topic]

```
</details>

**When to Use Each Level:**
- Judge by **number of deflections** and **tone intensity**
- Don't stay at Level 1 too long
- Always have Level 3 ready by deflection 5
